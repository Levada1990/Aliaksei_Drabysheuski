Дополнительные теги

Есть множество тегов, которые всегда пригодятся верстальщику в разработке, однако учить их не обязательно. Достаточно знать, где искать.

  • Теги <dl>, <dt>, <dd>
  • Тег <pre>
  • Теги <code>, <tt>
  • Тег <kbd>
  • Тег <var>
  • Теги <sub>, <sup>
  • Теги <blockquote>, <q>
  • Теги <ins>, <del>
  • Тег <address>
  • Теги <nobr>, <wbr>

Рассмотрим каждый тег в отдельности:

Теги <dl>, <dt>, <dd>

Используются для отображения списков определений.


Тег <pre>

Тег для отображения предварительно отформатированного текста.


Теги <code>, <tt>

Этот тег обычно используется для отображения кода. Однако тут следует учесть, что внутри данного тега пробелы и переносы не учитываются.
Поэтому обычно используют теги <br> и <p>.


marks

Тег <kbd>

Данный тен используется для текста, который набирают с клавиатуры или обозначения клавиш. Обычно данный текст после отображения в браузере становится моноширенным.


Тег <var>

Тег для обозначения переменных в тексте.


Теги <sub>, <sup>

Тег для обозначения нижнего и верхнего индексов соответственно.


Теги <blockquote>, <q>

Данные теги для обозначения длинных и коротких цитат соответственно.


Теги <ins>, <del>

Данные теги для обозначения вставленного и удаленного текста соответственно.


Тег <address>

Внутри данного тега обычно хранится информация об авторе текста. Как правило выделяется курсивом.


Теги <nobr>, <wbr>

Тег <nobr> дает команду браузеру отображать текст без переносов.


Тег <wbr> указывает место переноса в тексте. Часто используется для переноса длинных слов.
